### demo预览
- 下载demo代码后，打开index.html即可看到弹幕效果

### demo概要
- 1， 使用js随机生成颜色值、中文汉字、弹幕上下坐标、滚动速度等信息
- 2，可进一步封装成具体项目中所需要的弹幕效果

### demo优缺
* 优点
- 1，所有的弹幕动画使用css实现，性能上会好点
- 2，技术栈原生js，可进一步封装成为适用于项目需要的组件
* 缺点
- 1，目前PC端可覆盖在视频上层，移动端恐怕不能覆盖原生标签
- 2，弹幕的行数不能提前设置，随机生成
